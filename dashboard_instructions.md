# Dashboard Instructions

The `dashboard.xlsx` file is pre-generated by `create_dashboard.py` and includes two bar charts and a KPI. If you want to regenerate it or create the dashboard manually in Google Sheets, follow the steps below.

The processed CSV already contains the final fields needed for a small spreadsheet dashboard:

- `user_id`
- `name`
- `email`
- `city`
- `company`
- `total_posts`

## Google Sheets or Excel

1. Import `processed_users.csv`.
2. Create a tab named `Dashboard`.
3. Add a pivot table for users by city:
   - Rows: `city`
   - Values: count of `user_id`
4. Add a pivot table for posts by company:
   - Rows: `company`
   - Values: sum of `total_posts`
5. Add one KPI for the average posts per user:

```text
=AVERAGE(F2:F11)
```

## Visual organization

I kept the dashboard intentionally clean: a white background, simple blue headers, two charts side by side and one KPI at the top. This makes it look professional without overcomplicating the delivery.
